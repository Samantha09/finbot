# 广发 ETF 筛选 Skill

## 适用范围

当用户有以下意图时，必须使用 `gfEtfSearch` 工具：
- 找特定主题的 ETF（如"科技 ETF"、"新能源 ETF"）
- 按收益/风险条件筛选 ETF（如"近1月涨幅超 5% 的 ETF"）
- 按估值条件筛选（如"低估值的宽基 ETF"）
- 构建 ETF 候选池（如"推荐几个适合做网格交易的 ETF"）
- 查看 ETF 的实时涨跌幅、规模、PE/PB、指数温度

## 工具调用规则

### 1. 基础查询

| 用户意图 | 参数示例 | 说明 |
|----------|----------|------|
| 找行业 ETF | `trakType: "行业"` | 先定赛道大类 |
| 找宽基 ETF | `trakType: "宽基"` | 如沪深300、上证50 |
| 找主题 ETF | `trakType: "主题"` | 如人工智能、黄金 |
| 模糊搜索 | `search: "科技"` | 按名称或代码搜索 |
| 指定代码 | `tradeCode: "510050,510300"` | 多个逗号分隔 |

### 2. 收益与风险筛选

| 用户意图 | 参数示例 | 说明 |
|----------|----------|------|
| 近1月涨幅超 5% | `roc1m: "5~"` | `5~` 表示 >=5% |
| 近1年涨幅 0~20% | `roc1y: "0~20"` | 区间写法 |
| 回撤控制 | `maxDrawdown1y: "~20"` | 近1年回撤 <=20% |
| 夏普比率 | `sharpRatio1y: "1~"` | 近1年夏普 >=1 |

### 3. 估值与规模筛选

| 用户意图 | 参数示例 | 说明 |
|----------|----------|------|
| 低估值 | `valuationResult: "1"` | 1=低位 2=中位 3=高位 |
| 指数温度低 | `indexTempType: "low"` | low/ord/high |
| 大规模 | `assetScale: "10~"` | 规模 >=10亿 |

### 4. 交易属性

| 用户意图 | 参数示例 | 说明 |
|----------|----------|------|
| T+0 ETF | `tradeT0: "1"` | 跨境/商品/债券 ETF |
| 两融标的 | `marginTrade: "1"` | 可融资融券 |

### 5. 排序与分页

| 用户意图 | 参数示例 | 说明 |
|----------|----------|------|
| 按近1月涨幅排序 | `sort: "-roc1m"` | 降序加 `-` |
| 按规模排序 | `sort: "-assetScale"` | 从大到小 |
| 限制结果数 | `limit: 20` | 默认 20 |
| 获取实时数据 | `addRealTimeRoc: 1` | 推荐加上 |

## 常用组合示例

### 示例 1：近1月涨幅前 10 的行业 ETF
```json
{
  "trakType": "行业",
  "roc1m": "5~",
  "sort": "-roc1m",
  "limit": 10,
  "addRealTimeRoc": 1
}
```

### 示例 2：低估值 + 大规模的宽基 ETF
```json
{
  "trakType": "宽基",
  "valuationResult": "1",
  "assetScale": "50~",
  "sort": "-assetScale",
  "limit": 10
}
```

### 示例 3：夏普比率 > 1 且回撤 < 15% 的 ETF
```json
{
  "sharpRatio1y": "1~",
  "maxDrawdown1y": "~15",
  "sort": "-sharpRatio1y",
  "limit": 20
}
```

### 示例 4：T+0 跨境 ETF
```json
{
  "tradeT0": "1",
  "sort": "-roc1m",
  "limit": 15,
  "addRealTimeRoc": 1
}
```

## 输出规范

1. **结果展示**：使用 Markdown 表格展示代码、名称、赛道、涨跌幅、估值、规模
2. **数据时效性**：注明数据时间
3. **风险提示**：任何推荐末尾必须包含 `⚠️ 不构成投资建议`
4. **空结果处理**：如果无结果，建议用户放宽条件（如扩大涨跌幅区间、去掉赛道限制）

## 禁忌

- ❌ 禁止在没调用工具的情况下给出具体 ETF 推荐
- ❌ 禁止给出明确的买入/卖出建议
- ❌ 禁止泄露用户的筛选偏好作为投资建议
